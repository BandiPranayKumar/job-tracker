# 🎯 Job Application Tracker — Full Stack

A full-stack job application tracking system built with **Node.js + Express + MongoDB** (backend) and **HTML + CSS + JavaScript ES6 Modules** (frontend).

## Quick Start

### 1. Backend
```bash
cd backend
npm install
cp .env.example .env   # then edit .env with your MongoDB URI
npm run dev
```

### 2. Frontend
Open `frontend/index.html` with **VS Code Live Server**.

## Tech Stack
| Layer | Technology |
|-------|-----------|
| Frontend | HTML5, CSS3 (Flexbox/Grid), Vanilla JS (ES6 Modules) |
| Backend | Node.js, Express.js |
| Database | MongoDB + Mongoose |
| Auth | JWT + bcryptjs |
| Validation | express-validator |

## API
Base URL: `http://localhost:5000/api`
